Em uma indústria química, é conduzido determinado processo ao qual deve ser adicionado óleo quente. Para o condicionamento desse fluido, dispõe-se de um tanque acumulador agitado contendo uma camisa de aquecimento, conforme demonstrado na figura a seguir.

Inicialmente, esse tanque continha $m_0 = 50\ \mathrm{kg}$ de óleo a $20\ \mathrm{°C}$. Então, as linhas de alimentação e descarga foram abertas, o que permitia o fluxo de óleo à vazão de $3\ \mathrm{kg/s}$, mantendo-se, portanto, o nível do tanque constante. O coeficiente global de troca térmica, $U_C$ , sugerido pelo fabricante é de $200\ \mathrm{W \ m^{-2} \ K^{-1}}$, a área da camisa de vapor é de $15\ \mathrm{m^2}$ e a capacidade calorífica do óleo, $c_{p\mathrm{,óleo}}$ , é de $1{,}5\ \mathrm{kJ\ kg^{-1} \ K^{-1}}$.

O balanço de energia de um sistema sem reação, com variação desprezível de energia cinética e potencial e sem trabalho de eixo é dado por $$ \frac{\mathrm{d}U}{\mathrm{d}t} = \sum{ \dot{m_e} \ h_e } -\sum{ \dot{m_\mathrm{s}} \ h_\mathrm{s} } + \dot{Q} $$ em que: $U$ é a energia interna; $\dot{m}$, a vazão mássica; $h$, a entalpia por unidade de massa; $\dot{Q}$, a taxa de calor, definido por $\dot{Q} = U_C A (T_v - T_s)$; e os subscritos são $e$, para a corrente de entrada; e $s$, para a corrente de saída; $T_v$ é a temperatura do vapor; e $A$ é a área de troca térmica.

Com base nesses dados, assinale a opção que apresenta o valor aproximado da temperatura de saída do óleo após $100 \ \mathrm{s}$ de operação.

$$\frac{\mathrm{d}U}{\mathrm{d}t} = \sum{ \dot{m_e} \ h_e } -\sum{ \dot{m_\mathrm{s}} \ h_\mathrm{s} } + \dot{Q} $$

$$\frac{\mathrm{d}U}{\mathrm{d}t} = \dot{m_e} \ h_e -\dot{m_\mathrm{s}} \ h_\mathrm{s} + \dot{Q} $$

$$\frac{\mathrm{d}U}{\mathrm{d}t} = \dot{m} \ (h_e - h_\mathrm{s}) + \dot{Q} $$

$$\frac{\mathrm{d} (m_0 \ c_{p\mathrm{,óleo}} T) }{\mathrm{d}t} = \dot{m} \ c_{p\mathrm{,óleo}}(T_e-T_s) + \dot{Q} $$

$$\frac{\mathrm{d} (m_0 \ c_{p\mathrm{,óleo}} T_s) }{\mathrm{d}t} = \dot{m} \ c_{p\mathrm{,óleo}}(T_e-T_s) + U_C \ A \ (T_v-T_s) $$

$$\frac{\mathrm{d} (m_0 \ c_{p\mathrm{,óleo}} T_s) }{\mathrm{d}t} = \dot{m} \ c_{p\mathrm{,óleo}}(T_e-T_s) + U_C \ A \ (T_v-T_s) $$

$$ m_0 \ c_{p\mathrm{,óleo}} \frac{\mathrm{d} T_s }{\mathrm{d}t} = \dot{m} \ c_{p\mathrm{,óleo}}(T_e-T_s) + U_C \ A \ (T_v-T_s) $$

$$ m_0 \ c_{p\mathrm{,óleo}} \frac{\mathrm{d} T_s }{\mathrm{d}t} = (\dot{m} \ c_{p\mathrm{,óleo}} \ T_e + U_C \ A \ T_v) - (\dot{m} \ c_{p\mathrm{,óleo}} + U_C \ A) T_s $$

$$ \frac{\mathrm{d} T_s }{\mathrm{d}t} = \underbrace{\left(\frac{\dot{m} \ c_{p\mathrm{,óleo}} \ T_e + U_C \ A \ T_v}{m_0 \ c_{p\mathrm{,óleo}} } \right)}_{a} - \underbrace{\left(\frac{\dot{m} \ c_{p\mathrm{,óleo}} + U_C \ A}{m_0 \ c_{p\mathrm{,óleo}} }\right)}_{b} T_s $$

$$ \frac{\mathrm{d} T_s }{\mathrm{d}t} = a - b \ T_s $$

$$ \frac{\mathrm{d} T_s }{\mathrm{d}t} = a - b \ T_s $$

$$ \frac{\mathrm{d} T_s }{a - b \ T_s} = \mathrm{d}t $$

$$ \int_{T_{0}}^{T_s} {\frac{\mathrm{d} T'_\mathrm{s} }{a - b \ T'_\mathrm{s}}}= \int_{0}^{t}\mathrm{d}t' $$

$$-\frac{1}{b} \ \ln \left(\frac{a-b \ T_s}{a - b \ T_0}\right) = t$$

$$ T_s = \frac{-a + (a-b \ T_0)\exp(-bt)}{-b} $$

$$ T_s = \frac{a - (a-b \ T_0)\exp(-bt)}{b} $$

$$a = \frac{\dot{m} \ c_{p\mathrm{,óleo}} \ T_e + U_C \ A \ T_v}{m_0 \ c_{p\mathrm{,óleo}} }$$

$$a = \frac{3 \ \textcolor{LightCoral}{\mathrm{kg/s}} \times 1500\ \ \textcolor{LightCoral}{\mathrm{J\ kg^{-1} \ K^{-1}}} \times 20 \textcolor{LightCoral}{\mathrm{\degree C}} + 200 \ \textcolor{LightCoral}{\mathrm{W \ m^{-2} \ K^{-1}}} \\ \times 15 \ \textcolor{LightCoral}{\mathrm{m^2}} \times 120 \textcolor{LightCoral}{\mathrm{\degree C}}}{50 \ \textcolor{LightCoral}{\mathrm{kg}} \times 1500 \ \textcolor{LightCoral}{\mathrm{J\ kg^{-1} \ K^{-1}} }}$$

$$a = 6 \ \textcolor{LightCoral}{\mathrm{J/s}}$$

$$b = \frac{\dot{m} \ c_{p\mathrm{,óleo}} + U_C \ A}{m_0 \ c_{p\mathrm{,óleo}} }$$

$$b = \frac{3 \ \textcolor{LightCoral}{\mathrm{kg/s}} \times 1{,}5\ \ \textcolor{LightCoral}{\mathrm{kJ\ kg^{-1} \ K^{-1}}} + 200 \ \textcolor{LightCoral}{\mathrm{W \ m^{-2} \ K^{-1}}} \times 15 \ \textcolor{LightCoral}{\mathrm{m^2}}}{50 \ \textcolor{LightCoral}{\mathrm{kg}} \times 1{,}5 \ \textcolor{LightCoral}{\mathrm{kJ\ kg^{-1} \ K^{-1}} }}$$

$$b = 0{,}1 \ \textcolor{LightCoral}{\mathrm{s^{-1}}} $$

$$ T_s = \frac{a - (a-b \ T_0)\exp(-bt)}{b} $$

$$ T_s = \frac{6 \ \textcolor{LightCoral}{\mathrm{J/s}} - (6 \ \textcolor{LightCoral}{\mathrm{J/s}}-0{,}1 \ \textcolor{LightCoral}{\mathrm{s^{-1}}} \times 20 \textcolor{LightCoral}{\mathrm{\degree C}})\exp(-0{,}1 \ \textcolor{LightCoral}{\mathrm{s^{-1}}} \times 100 \ \textcolor{LightCoral}{\mathrm{s}} )}{0{,}1 \ \textcolor{LightCoral}{\mathrm{s^{-1}}}} $$

$$ T_s \approx 60 \textcolor{LightCoral}{\mathrm{\degree C}} $$
